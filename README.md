
설명: 생성형 AI 모델이 제작한 가상 비디오(Synthetic Video)와 실제 촬영 영상을 무작위로 배치하여 인간의 인지적 판별력을 테스트하고, 이에 따른 정답률 및 오탐률 데이터를 수집·분석하는 실험 플랫폼입니다.

주요 기능: 시각 자극(영상) 제시, 사용자 응답 수집, 데이터 정량화 및 엑셀(CSV) 내보내기.

인간 인지 기반 평가(Perceptual Evaluation): 컴퓨터 비전 및 생성 모델 연구에서 필수로 요구되는 '사용자 평가(User Study)' 파이프라인을 직접 기획하고 구현했습니다. AI 생성물의 정교함을 사람의 인지 관점에서 정량적 데이터로 추출하는 환경을 구축해 보았습니다.

데이터셋 정제 능력: 피실험자들의 응답을 정답률과 오탐률(False Alarm Rate) 지표로 세분화하여 분석 가능한 엑셀 데이터 형태로 구조화했습니다. 연구실에서 개발한 생성 모델의 시각적 퀄리티를 검증하는 인간 중심 벤치마크 툴로 확장 가능합니다.
